Classroom Reservation System (JavaFX)

A desktop application built with JavaFX to simplify classroom scheduling. Admins manage rooms and reservations, while users search availability, submit requests, and track approvals to avoid double booking.

  • Admin panel (room and reservation management)
  • User panel for availability search and booking
  • Approval flow prevents conflicts for meetings and events

KFUPM Events Hub

A centralized event platform for KFUPM. Organizers create events, admins approve them, and students discover, register, and get reminders in one place instead of scattered channels.

  • Admin approval flow to publish events safely
  • Student feeds for suggested and recent events
  • QR check-in and attendance analytics

Study Companion (Web)

A browser-based assistant that tracks study sprints and checklists. It stores daily focus goals locally, surfaces motivation quotes, and uses timers to keep me on-task between classes.

  • Pomodoro-style timer with pause and resume
  • LocalStorage for focus goals and quick reloads
  • Motivation feed powered by Quotable API
